# Personal Intelligence Briefing

个人智能情报与持续学习系统。

MVP 聚焦 AI 行业实时信息：GitHub Actions 维护可追溯的候选事件与去重状态，ChatGPT 计划任务完成最终审稿与定向查漏。

## 安全边界

- 仅处理公开互联网信息。
- 不提交 API Key、Cookie、个人隐私、公司或客户资料。
- 模型密钥仅从 GitHub Actions Secrets 读取。
- Markdown 是 MVP 的唯一交付格式；PDF、Word 与网页阅读页不在首阶段范围内。

详细产品设计见 `docs/superpowers/specs/`。
